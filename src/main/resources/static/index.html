1. Загрузить файл 2. Скачать файл 3. Конвертация 4. Уникальная конвертация

Легенда кнопок:

  1. Загрузить файл на сервер.
  2. Скачать файл с сервера, если такой существует.
  3. Конвертировать файл, который существует на сервере, скачать результат.
  4. Загрузить файл, конвертировать и скачать на лету.
  5. Посмотреть все файлы, которые доступны для скачивания.
Конвертация происходит из XML в JSON и наоборот, не забывайте писать расширение в названии файлов.
*Рекомендуется использовать уникальную конвертацию, поскольку результат будет получен сразу.
*Файлы загруженные на сервер будут удалены в 14:30 15-го числа каждого месяца.
